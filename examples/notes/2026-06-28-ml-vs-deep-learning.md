# Machine Learning vs Deep Learning

## In one line
ML learns from data with human-defined features; deep learning is a subset that uses multi-layer neural nets to learn features itself from raw data.

## Key points
- **ML:** learns from historical data without explicit programming; humans usually hand-pick the "features." Works on smaller/structured data (spreadsheets).
- **ML uses:** spam filtering, house-price prediction, recommendations (Netflix/Amazon), fraud detection.
- **Deep learning:** brain-inspired neural networks that **auto-extract features** from raw, unstructured data (images, text, audio) — no manual feature engineering.
- **DL cost:** needs *massive* data and heavy compute to train well. DL is the more powerful but hungrier branch of ML.

## Why I saved this
The clean distinction (manual features + small data vs auto features + big data) I keep needing when explaining where DL fits.
