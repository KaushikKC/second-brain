# Examples

These are real notes produced by the system from a single pasted study document about LLM/AI
fundamentals. They show:

- the **note format** in practice (one-liner, 3–5 key points, why-saved, source),
- how one dense capture gets **split into several short, linked notes** instead of one long one,
- how notes **cross-link** with `[[wikilinks]]` to form a small knowledge graph.

They're here for reference only — your own notes will land in `notes/`. Feel free to delete this
folder once you've seen the style.
